submission guidelines
Guidelines as of 14th Oct 2019
1- All posts must be clearly related to Svelte. If you only make a passing comment, avoid using the tag.
2- You needn't be a master of the framework in order to post. Any level of knowledge and skill is acceptable.
3- Feel free to post all sorts of stuff (tutorials, questions, thoughts, critiques, etc.) as long as you stay on topic.
about #svelte
Svelte is a component framework for JavaScript that runs at build time. It turns components into imperative code and forgoes virtual DOM.
