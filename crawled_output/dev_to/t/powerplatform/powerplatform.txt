Power Platform is Microsoft's low code platform and consists of the following systems
Power Automate - cloud and desktop based automation tool
Power Apps - business application builder
Power Virtual Agents - chatbots
Power Pages - external facing websites
Power BI - dashboard and analytics
Dataverse - database
Power FX - expression based coding language
AI Builder - Character & Image recognition, translation and language model
