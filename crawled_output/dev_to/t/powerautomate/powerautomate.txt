Power Automate - Microsoft Power Platform
Follow
Hide
Power Automate is a cloud and desktop based automation tool. Part of Microsoft's Low Code Power Platform. The Cloud version is based on Azure Logic-Apps, allowing you to connect different API's using wrappers called connections, along with the ability to be triggered by other systems. These connectors are provided by Microsoft (Graph API based) and external companies/independent developers.
The desktop version is an rpa tool, allowing automation of websites and standard applications.
