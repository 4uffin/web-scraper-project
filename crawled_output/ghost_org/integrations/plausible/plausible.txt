AnalyticsWant simple, privacy-focused analytics for your Ghost publication? Integrate with Plausible using straightforward code injection.Plausible is a privacy-first analytics platform that delivers insights in clear and extremely simple graphs without compromising user privacy. With a few easy steps you can enable this for your Ghost site and start gaining valuable insights about your visitors right away.Add your website to PlausiblePlausible offers a free trial which you can use to get started. Follow the instructions to create your account. You will be asked to add the website you want to keep track of and select your timezone:Get the tracking codeCopy the javascript snippet from the following page and go to your Ghost Admin page:Use Ghost code injectionIn Ghost you can inject code across your entire site or on an individual post or page. Since Plausible needs to track analytics across your entire site, we’ll use the global code injection feature which can be found in the Ghost Admin settings menu.Plausible requires the tracking code to be in the <head> of each page on your site, so paste it into the Header section and hit save.All done! Your site is now fully integrated with Plausible.
