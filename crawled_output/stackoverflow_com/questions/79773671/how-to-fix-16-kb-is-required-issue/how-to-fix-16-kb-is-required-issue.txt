One of your Gradle dependencies itself depends on something that contains this library. Make sure that you are up to date on all of your direct dependencies, and hopefully the latest versions will clear up the problem. If not, look for dependencies that suggest that they use AI.

You can update/migrate the tensor flow libs as below ->
Old
implementation 'org.tensorflow:tensorflow-lite-support:0.x.0'
implementation 'org.tensorflow:tensorflow-lite-metadata:0.x.0'
to
New
implementation 'com.google.ai.edge.litert:litert-support:1.4.0'
implementation 'com.google.ai.edge.litert:litert-metadata:1.4.0'
