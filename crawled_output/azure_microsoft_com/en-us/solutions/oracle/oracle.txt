FAQ
Frequently asked questions
Expand all
Collapse all
01/
Where is Oracle Database@Azure available?
Oracle Database@Azure is available in more regions than any other hyperscaler, with plans to expand to additional regions in 2025 (based on customer demand). If you’re interested in learning more about availability, please contact your respective Microsoft or Oracle account team.
02/
Which Oracle services will be available in Azure?
Oracle will build, deploy, and operate a set of Oracle Database Services in Azure that were previously available only on-premises or in Oracle Cloud Infrastructure (OCI).
The initial offering included Oracle Exadata Database Service and support for Oracle Real Application Clusters, Oracle GoldenGate, and Oracle Data Guard technologies. We've since expanded the offering to include Oracle Autonomous Database, Oracle Database Zero Data Loss Autonomous Recovery Service, Oracle Exascale, and Base Database (preview)
03/
How can I buy Oracle Database@Azure?
Oracle Database@Azure is offered through Azure Marketplace. You can take advantage of existing Azure and Oracle licenses and commitments. Contact your Microsoft account team to learn more.
Go to Azure Marketplace
04/
What is the best way to get expert help, investments and resources to start your cloud journey with Azure?
Get guidance, browse resources, and find expert help for your move to Azure. Learn more about Azure Accelerate.
05/
Do you offer Oracle Interconnect for Azure?
Azure was the first cloud provider to partner with Oracle and launch an OCI Interconnect solution in 2019. While the solution is still offered, consider our newest joint solution Oracle Database@Azure for your mission-critical workloads.
06/
Are Oracle apps supported in Azure?
Azure offers a set of services for modernizing both apps and databases. You can use your favorite tools and frameworks for app development and deployment. Easily add services and capabilities, including managed Azure databases, to modernize or develop enterprise applications.
