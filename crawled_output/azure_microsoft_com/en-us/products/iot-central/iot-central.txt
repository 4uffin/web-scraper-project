Evaluate your IoT projects for business value and viability
Azure IoT Central is a ready-made UX and API surface for connecting and managing devices at scale, delivering reliable data for business insights. It preassembles platform as a service (PaaS) offerings, bringing together each service beneath it for an easy-to-configure, comprehensive, and secure IoT offering.
Quick connectivity between IoT devices and the cloud
Centralized management to easily reconfigure and update devices
Visualizations and analysis to make sense of IoT data, from the big picture to small details
Extensibility to bridge business applications and IoT data, turning insights into action
Evaluate your IoT project with Azure IoT Central
Get started using a ready-made web experience and API surface for IoT solution development. Manage devices at scale using the job scheduler. Store data over a 30-day window, visualizing in custom dashboarding and data exploration experiences. Define rules to trigger business flows, send IoT data to other services using data export, and programmatically access Azure IoT Central using its public REST API.

Frequently asked questions about Azure IoT Central
Expand all
|
Collapse all
How can I evaluate IoT projects using Azure IoT Central?
IoT Central is the fastest and easiest way to evaluate your IoT scenario and assess the opportunities it can create for your business. Learn more.
Does Azure IoT Central have high availability and disaster recovery (HADR) and elastic scale?
Yes, Azure IoT Central is an application PaaS that manages scalability and HADR for you. An Azure IoT Central application can scale to support millions of connected devices. Learn more.
Can I extend Azure IoT Central?
Yes, Azure IoT Central has a rich set of extensibility features, including rules to trigger business flows, data export to send IoT data to other services, and a public REST API. Learn more.
What are the scale limits?
Various quotas and limits apply to Azure IoT Central applications. View the quotas and limits documentation.
What’s the service level agreement?
Azure IoT Central gives you guaranteed 99.9-percent connectivity. Read the SLA.
What’s the data retention policy?
Azure IoT Central data is retained on a rolling 30-day basis. You can continuously export data using the export feature.
