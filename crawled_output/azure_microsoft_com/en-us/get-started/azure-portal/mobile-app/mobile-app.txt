Get the Azure mobile app
Stay connected to your Azure resources—anytime, anywhere.
Monitor resources and get Azure Service Health alerts
Keep track of the status of your Azure resources, such as virtual machines (VMs) and web apps, from your mobile device. And get alerts on your environment.
Quickly diagnose and fix issues
Respond quickly to alerts, view metrics, and take corrective actions. Restart services like App Service web apps or connect to a VM directly.
Manage your resources through Azure Cloud Shell
Use Azure CLI or PowerShell commands to manage your Azure environment. Work in Bash or PowerShell through Cloud Shell and persist files across interfaces.
See how the Azure mobile app works
Watch the video on how to use the Azure mobile app to monitor and manage your Azure resources. Learn how to take control of your environment from the convenience of your phone.

Azure mobile app FAQs

How do I sign into the Azure mobile app?
Use your Azure account to sign in. If you don’t have one, create an Azure free account.
Where can I download the Azure mobile app?
Download and install the Microsoft Azure app for free from Google Play and the Apple App Store.
Which versions of iOS and Android does the Azure mobile app support?
The Azure mobile app supports iOS 11.0 and later and Android 6.0 and later.
Which languages does the Azure mobile app support?
The app supports the following languages: Simplified Chinese, Traditional Chinese, Czech, Dutch, English, French, German, Hungarian, Italian, Japanese, Korean, Polish, Portuguese, Russian, Spanish, Swedish, and Turkish.
Does the Azure mobile app support tablets?
The app is optimized for iOS and Android smartphones but works on tablets as well.
How can I sign up for an Azure account so I can start using the mobile app?
Create an Azure free account to get started. The Azure free account includes popular services free for 12 months, more than 25 other services that are free always, and $200 credit to use in your first 30 days.
