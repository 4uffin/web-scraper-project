Instructions to run gridfinder on cloud through SSH
Instructions to run gridfinder on cloud through SSH
Open your terminal and type the following and enter the password.
$ ssh <user>@<IP>
There are several different processes running in different tmux sessions. I prepared two sessions called g1 and g2 that we can use to run the algorithm in parallel. To connect to one of the sessions, type the below. There should now be an extra bar on the bottom of the terminal with the session name in it.
$ tmux attach -t
