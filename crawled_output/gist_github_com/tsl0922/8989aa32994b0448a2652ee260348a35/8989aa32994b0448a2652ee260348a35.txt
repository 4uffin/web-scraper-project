UPDATE: You may use dialog.lua for clipboard and file dialog now.
why is there no open file and Open URL or file from clipboard?
It's not supported by mpv yet, you need other scripts:
clipboard: https://github.com/jonniek/mpv-scripts#appendurllua
file dialog: https://github.com/rossy/mpv-open-file-dialog
recent: https://github.com/natural-harmonia-gropius/recent-menu
Change the last line of appendURL.lua to mp.add_key_binding('', 'appendURL', append)
Change the last line of open-file-dialog.lua to mp.add_key_binding('', 'openFileDialog', open_file_dialog)
Add the following lines to the beginning of the input.conf above:
Ctrl+o
script-binding openFileDialog
#menu: Open > Files...
Ctrl+v
script-binding appendURL
#menu: Open > Clipboard
_
script-binding recentmenu/open
#menu: Open > Recent #@recent
_
ignore
#menu: -
Now you can open file dialog via Open > Files... menu or Ctrl+o hotkey, open clipboard via Open > Clipboard menu or Ctrl+v hotkey.
