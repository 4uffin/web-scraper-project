xfce-i3-guide.md
How to XFCE with i3WM
Download
Download XFCE Distribution
Download i3 only (i3 status i3 lock dmenu is not needed)
⚠️ Do not installed preconfigured or distributed i3wm to avoid conflicts with XFCE settings
e.g.
do sudo dnf install i3 --setopt=install_weak_deps=False
dont "i3 desktop" or @i3-desktop-environment
Add i3WM Startup
Open Session and Startup
Select Application Autostart
Add
Name: i3
Description: Window Manager
Command: i3
Trigger: On login
Name: xfsettingsd
Description: Xfwm Theme Workaround
Command: xfsettingsd --replace
Trigger: On login
Disable XFCE Application Startup
Edit Current Session
Program: xfdesktop
Restart Style: Never
Program: xfwm4
Restart Style: Never
Restart
Post configuration
Remove all XFCE keyboard shortcuts through XFCE Keyboard
Change theme through XFCE Appearance
Install lightdm-gtk-greeter-settings to change login screen theme
Change wallpaper through XFCE Desktop
Bind Print Screen button to xfce4-screenshooter (.config/i3/config)
Install picom if you are experiencing screen tearing (.config/picom.conf)
